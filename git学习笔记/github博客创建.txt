1.在github上创建阻止

2.github上如jquery开源项目，找到可以看到它不是在某个人的下面创建的jquery仓库，而是以团队或者阻止创建的

3.点击github上个人主页 左上角的+ 选择New organization 创建组织


4.（1）按照要求填写即可  有公开或私有（花钱），点击按钮完成创建

（2）第二步 点击添加那些开发人员或者你这个组织的开发人员

（3）点击finish 完成 后出现一个页面 ，你可以设置或者浏览对应信息

（4）在右边暂时只显示一个人 你自己  ，你添加的团队成员，必须等他们同意才能现在在这个地方

在github上创建博客

　　　　创建博客
　　　　github上创建
　　　　https://pages.github.com/
　　　　注意格式的正确性

1.https://pages.github.com/  这个就是专门介绍如何创建博客的 按照步骤即可


2.创建博客

（1）登录自己的主页，点击创建仓库

（2）此时仓库的名字和你账户的名字是相同的

（3）写上创建的信息，别忘记点击创建readme.md文件

（4）在git上开发博客也可以，在github上开发也可以

（5）点击New file ----------  我在github上创建一个index.html首页 

3.https://alingfeng.github.io/ 就可以访问博客了  

4.建议：不要在github上编写博客，因为有的格式不支持，导致布局错位
